## Discoveries (fixes that took multiple cycles)

1. **Workflow-lint regex bug** (`re.escape` missing on `$`) — silently
   broke workflow-lint for 2 weeks.
2. **Branch protection required check named `validate`** — no workflow
   had a job by that name. Created `validate.yml`.
3. **Godot import timeout** — 180 s was insufficient for 206 cold
   .ctex generations on ubuntu-latest. Bumped to 360 s.
4. **`.import` files generated by Python** had wrong hash format —
   Godot regenerated all 257 correctly on next CI run.
5. **Playtester `create_timer(SHOT_INTERVAL)` ran AFTER
   `Engine.time_scale = 8.0`** — each tick fired in 0.3 s real time
   (2.5 ÷ 8). PR #433 fixes with `ignore_time_scale=true`.
6. **Dev menu grey screen root cause** — extra `)` at line 1040 of
   `dev_menu.gd` (`spawn_step_dust(center))`) + missing helper functions
   `_count_tres` / `_count_pngs` referenced from Build Info tab. PR
   #421 fixes both.
